The code is written in python.
To run the code simply type: 
	python main.py 
in the terminal. 
Please navigate to main.py folder and make sure the gloss.py and manga.py are included in the folder
The script requires the input images to come from ./pics/source/test_dir/ 
The resulting output will go into ./pics/out/test_dir/
The user can go into the main.py script to change directories.
